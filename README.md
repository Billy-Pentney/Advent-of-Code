# Advent of Code (AoC)

## Description

Collection of my solutions (some partial) to Advent of Code problems.

Written in Python because it's easy and quick.

## Template Creation

Use the `makeday.py` script to create new blank solutions, e.g. 

    python makeday.py -y 2025 -d 1 4 5

would make the following files from the template: 

    2025/01
        01.py
        example.txt
        test.txt
    2025/04
        04.py
        example.txt
        test.txt
    2025/05
        05.py
        example.txt
        test.txt
